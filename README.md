# Checkout E2E Automation

Cypress E2E tests for Circuly Checkout with multiple shop platforms and payment gateways.

## Supported Platforms

- Shopify
- Saleor
- WooCommerce
- Shopware5

## Supported Payment Gateways

- Stripe (Card, SEPA, BACS)
- Mollie
- Adyen
- Braintree

## Quick Start

```bash
# Install dependencies
npm install

# Copy environment template
cp .env.example .env

# Open Cypress
npm run cy:open
```
